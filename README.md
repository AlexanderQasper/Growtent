# Growtent

Modular control system for an indoor grow tent based on Arduino and Raspberry Pi.

## Features
- Air temperature, humidity and pressure monitoring (BME280)
- Light level measurement
- Relay control (lighting / ventilation)
- Modular architecture (sensors, control, actuators, reporting)

## Tech stack
- Arduino (Uno / Mega)
- C++ (Arduino framework)
- Raspberry Pi (planned integration)

## Project status
Prototype (stage 1): sensors + relay control + basic automation

## Next steps
- Add hysteresis to control logic
- Implement automatic watering
- Integrate Raspberry Pi for monitoring and UI
